# 快速参考卡

> 各 skill 直接复制此模板，填入对应阶段、动作、确认节点即可。

## 通用格式

| 阶段 | 核心动作 | 确认节点 |
|------|---------|---------|
| Step X | [动作] | [确认方式] |

## 确认节点类型

| 类型 | 说明 |
|------|------|
| `AskUserQuestion` | 必须显式提问确认 |
| — | 自动执行，无需确认 |
| `PlanMode` | 复杂决策进入计划模式 |
